# 📱 NFC-Based Smart Attendance System

A modern, frictionless attendance tracking system designed to eliminate paper waste, save lecture time using Near Field Communication (NFC) technology. 

Developed by: Ahmad Albraa Alhumedy  
 Prince Sattam Bin Abdulaziz University  
Supervised by: Dr.Abdullah Albanyan

## ✨  Features
- **Frictionless Check-In/Out:** Students simply tap their registered smartphone on the classroom NFC reader.
- **Anti-Fraud Security:** Hardware ID locking ensures one student = one device.
- **Smart 75% Rule Engine:** Automatically calculates session duration and applies academic attendance policies dynamically.
- ** Dashboards:** Dedicated, role-based UI for Students, Instructors, and IT Administrators.
